GitHub Classroom Submission Guide

Step-by-step instructions for submitting assignments using GitHub Classroom

Getting Started

1. Accept the Assignment

Click your assignment link from Piazza/email:

https://classroom.github.com/a/AbC123

2. Create GitHub Account (First Time Users)

If you don't have a GitHub account:

  • Username: FirstName-LastName (e.g., Jane-Smith-SUNY)
  • Email: Your school email
  • Password: Create a strong, unique password

Note: Verify your email address by clicking the confirmation link sent to your inbox.

3. Link Your Student ID

After signup, you'll see:

Enter your university ID exactly as provided (e.g., SHAB12345)

Important: Use the exact ID your professor has on record!

Submitting Your Assignment

4. Access Your Private Repository

After accepting the assignment, you'll be redirected to your personal submission repo:

https://github.com/dmath-fall25/departure-pass-0-YourUsername

5. Upload Your Assignment Files

  1. Click "Add file" â†’ "Upload files"
  2. Drag & drop your PDF/ZIP file
  3. Rename files correctly: LASTNAME_AssignmentName.ext (e.g., SMITH_DeparturePass0.pdf)
  4. Scroll down and click "Commit changes"

Pro Tip: Submit at least 1 hour before deadline to avoid last-minute issues!

6. Verify Your Submission

Confirm your file appears in the repository with the correct name:

You'll see a timestamp confirming your submission time.

Deadline Information

Important Time Zone Notice

GitHub shows all deadlines in UTC (not EST):

  • Assignment Due: Sep 2, 11:59 PM EST
  • GitHub Displays: 03:59 AM UTC Sep 3

Convert to your local time: Time Zone Converter

Troubleshooting

Problem Solution
"Repository not found" Re-click the original assignment link
Can't upload files Check file size (<100MB) - compress to ZIP if needed
No "Commit" button Add a commit message at the bottom of the page
Uploaded wrong file Re-upload the correct file - it will overwrite the previous version
Forgot GitHub password Use GitHub password reset

After Submission

  • Professor will add feedback as annotated PDFs or FEEDBACK.md files
  • Check your email for GitHub notifications
  • Access feedback in your repository

Need help? Post privately on Piazza: #tech-support or tag @professor
