Required Software

You’ll need to use the LaTeX typesetting system to type your homework assignments this semester.

There are two main ways to get started:

  • Option 1 (Recommended for Beginners): Use Overleaf.com, a free, web-based LaTeX editor. No installation required—just create an account and start typing.
  • Option 2 (Install Locally): Set up LaTeX on your own computer by installing:
    1. A LaTeX distribution:
      • MacTeX (for macOS)
      • MikTeX (for Windows)
    2. A LaTeX editor: I recommend TeXstudio. You may also use the default editors included with the distributions:
      • TeXShop (included with MacTeX)
      • TeXworks (included with MikTeX)

Once your document is written and saved as a .tex file, you can compile it into a PDF.

More details and setup instructions will be provided in Lesson 0.
