A note about Problem Set 0

You may have noticed that the Knights and Knaves problem appears in both Departure Pass 2 and Problem Set 0. This is intentional!

For Departure Pass 2, a concise, intuitive explanation is perfectly acceptable.

For Problem Set 0, however, I would like you to present a more formal, complete analysis. Please model your solution after the methods we used in class: either by listing all possible scenarios (Knight/Knave combinations for A and B) and demonstrating which ones are logically consistent, or by constructing a truth table that formally validates your conclusion.

The goal is to ensure you are comfortable translating these logic puzzles into a structured, mathematical argument. This will help deepen your understanding of logical consistency and proof techniques.
