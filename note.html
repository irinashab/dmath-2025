A New Approach to Our Learning Environment

After careful consideration, I have decided not to use Brightspace for this course. This decision is based on a combination of aesthetic, moral, socio-psychological, and pedagogical reasons. Its sole function will be to host a single link to our primary hub: the lecture notes on GitHub Pages.

Instead, our coursework will be centered on three main platforms:

  1. GitHub: All lecture notes will be hosted publicly on GitHub Pages. Furthermore, we will use GitHub Classroom to manage the submission of your assignments and to provide my feedback. This platform offers a more transparent and efficient workflow for hands-on learning.
  2. Piazza: We will use Piazza to create a collective "class brain." This is the designated space to ask and answer all questions about course material, fostering collaborative learning and ensuring everyone has access to the same information.
  3. Student-Led Discussion: If you prefer an additional, more informal discussion space (e.g., Discord), I support you in setting it up and managing enrollment. Please feel free to organize this amongst yourselves.

I have also redesigned the grading approach to be more reflective of your learning journey. All details regarding these new methods are thoroughly documented within the lecture notes.

I am fully aware that this shift places us all in a slightly new territory. I am a novice in using GitHub as a classroom tool, and you are my partners in this experiment. I am confident that by putting our efforts together, we can not only manage but thrive in this new environment. I believe the skills we gain through this process will be incredibly valuable.

I am excited to embark on this collaborative learning journey with you all.
